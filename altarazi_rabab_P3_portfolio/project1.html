    New Tinder Feature

    For this project, me and my teammates Ana Lizarraga and Jessica Li added a new feature for the Tinder App.

    1.

    First we started by researching our target users and their needs to create a persona that would help us brainstorm ideas for a new feature for Tinder. We conducted interviews with many Tinder users and collected data that we can use and created a user persona

    2.

    Based on the data we collected we decided to work on a new feature called Date Request, in which the user can request to go on a date with their existing match. The feature allows users to plan time, activity and place for the date in a fun and an interactive way.<

    3.

    Finally, we created the final mockup for the new feature using Figma. We first created wireframes to test the flow of the interaction, and then we were happy with the result, we refined it to match Tinder's style guide and standards. After that, we created a prototype to be tested by users.
